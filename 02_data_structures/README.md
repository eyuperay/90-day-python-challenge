### 90-Day Python Challenge
This repository contains all Python exercises completed during a 90-day intensive Python learning program.

### 02_data_structures.py

**Description:**  
This script simulates a simple inventory system for a tech store using Python data structures such as lists and dictionaries.

**Key Features:**
- List operations (append, delete, slicing)
- Dictionary usage for stock management
- Loop to calculate total stock
- List comprehension for filtering data
